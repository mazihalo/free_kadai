# To-Doリスト

概要
To-Doリストは，コマンドからタスクの追加、表示、完了、削除が簡単に行えます。

機能
タスクの追加 (add)
タスクのリスト表示 (list)
タスクの完了 (complete)
タスクの削除 (delete)

使用方法
タスクの追加:

bash
コードをコピーする
node index.js add "タスク内容"
タスクのリスト表示:

bash
コードをコピーする
node index.js list
タスクの完了:

bash
コードをコピーする
node index.js complete 1
タスクの削除:

bash
コードをコピーする
node index.js delete 1
